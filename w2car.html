WEB 2.0


Caracteristicas

  • Representa la evolución de las aplicaciones tradicionales WEB enfocadas al usuario final.

  • En diferencia a la WEB 1.0, la WEB 2.0 es mas dinámica y mas atractiva para el usuario

  • Integran recursos multimedia como videos y sonidos.

  • Los contenidos de las diferentes páginas WEB pueden ser compartidas entre los usuarios.

  • La informacion que puedes encontrar puede ser escrita como también audiovisual.

  • Permite que el ususario cree su propio contenido es decir, el usuario puede comentar dichas páginas WEB.

  • Se utilizan las redes sociales para el compartir de los contenidos.

  • Proveen APIS o XML para que los usuarios puedan manipular siertas aplicaciones.

  • Facilita la localización de dicha página WEB por medio de los URL
